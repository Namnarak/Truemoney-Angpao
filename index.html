สวัสดีค่าาา ✦
ผู้บริการ API p.namnarak  — ยินดีต้อนรับสู่ TrueMoney Angpao Gateway ค่ะ
📬 รับเงินจากลิงค์ซอง — ใช้ได้เลย ไม่ต้องล็อกอิน

API นี้ช่วยให้คุณรับเงินจากลิงค์ซองของขวัญ TrueMoney ได้โดยตรง แค่ส่ง ลิงค์ซอง และ เบอร์โทร ของคุณ ระบบจะเคลมให้แบบเรียลไทม์เลยค่ะ

🔹 GET  — เรียกผ่าน URL โดยตรง
🔗 Endpoint GET
GET https://tmn-angpao.pages.dev/api/ {phone} / {voucher_id}
📋 พารามิเตอร์ (GET)
phone
เบอร์โทรศัพท์ 10 หลัก ของผู้รับเงิน
เช่น 0812345678
voucher_id
ลิงค์ซองของขวัญ เฉพาะส่วนหลัง ?v=
เช่น abc123xyz จาก https://gift.truemoney.com/campaign/?v=abc123xyz
📌 ตัวอย่าง (GET)
# ใช้ curl curl "https://tmn-angpao.pages.dev/api/0812345678/abc123xyz" # หรือเปิดในเบราว์เซอร์ https://tmn-angpao.pages.dev/api/0812345678/abc123xyz
🔸 POST  — ส่งเป็น JSON ใน Body
🔗 Endpoint POST
POST https://tmn-angpao.pages.dev/api
📋 Request Headers (POST)
Content-Type: application/json  ← จำเป็น
Accept: application/json  ← แนะนำ
📦 Request Body (JSON)
// Content-Type: application/json { "phone": "0812345678", "voucher_id": "abc123xyz" }
📌 ตัวอย่าง (POST)
# ใช้ curl curl -X POST "https://tmn-angpao.pages.dev/api" \ -H "Content-Type: application/json" \ -H "Accept: application/json" \ -d '{"phone":"0812345678","voucher_id":"abc123xyz"}'
📤 Response Headers
Content-Type: application/json; charset=utf-8
Cache-Control: no-store, no-cache, must-revalidate
X-Request-ID: a1b2c3d4e5f6  ← ใช้อ้างอิงตอนแจ้งปัญหา
Retry-After: 30  ← เฉพาะเมื่อ status = 429

💡 ทุก Response จะมี X-Request-ID สำหรับใช้ในการแจ้งปัญหาหรืออ้างอิง Log

📤 ตัวอย่าง Response
✅ สำเร็จ (200)
{ "success": true, "status": 200, "message": "รับเงินสำเร็จ", "code": "SUCCESS", "data": { "name": "นาย ทดสอบ ***", "amount": "10.00" } }
⏳ Rate Limited (429)
Header: Retry-After: 30
{ "success": false, "status": 429, "message": "ยิงคำขอเร็วเกินไป กรุณารอสักครู่", "code": "RATE_LIMITED", "data": null }
❌ ตัวอย่าง Error (400)
{ "success": false, "status": 400, "message": "ลิงค์ซองของขวัญหมดอายุ", "code": "VOUCHER_EXPIRED", "data": null }
📊 HTTP Status Summary
200 — Success ✅
400 — Bad Request
404 — Not Found
405 — Method Not Allowed
415 — Unsupported Media Type
429 — Too Many Requests
500 — Internal Server Error
⚠️ ตาราง Error Codes
HTTP Status Code ข้อความ
200 SUCCESS รับเงินสำเร็จ ✅
400 VOUCHER_EXPIRED ลิงค์ซองของขวัญหมดอายุ
400 VOUCHER_ALREADY_USED ลิงค์ซองของขวัญถูกใช้งานแล้ว
400 SELF_CLAIM_NOT_ALLOWED ไม่สามารถใช้อั่งเปาของตัวเองได้
400 INVALID_PHONE_OR_USED เบอร์รับเงินนี้ไม่ถูกต้อง หรือรับเงินไปแล้ว
400 INVALID_PHONE เบอร์รับเงินไม่ถูกต้อง (ต้อง 10 หลัก ขึ้นต้นด้วย 0)
400 INVALID_VOUCHER กรุณาระบุลิงค์ซองของขวัญ
400 MISSING_PARAMETER กรุณาระบุ phone และ voucher_id
400 INVALID_PATH รูปแบบ URL ไม่ถูกต้อง ใช้ /api/{phone}/{voucher_id}
400 INVALID_JSON JSON payload ไม่ถูกต้อง
404 VOUCHER_NOT_FOUND ไม่พบลิงค์ซองของขวัญ
405 METHOD_NOT_ALLOWED Method Not Allowed (ใช้ GET หรือ POST เท่านั้น)
415 UNSUPPORTED_MEDIA_TYPE Content-Type ต้องเป็น application/json (สำหรับ POST)
429 RATE_LIMITED ยิงคำขอเร็วเกินไป กรุณารอสักครู่ (Retry-After: 30)
500 INTERNAL_SERVER_ERROR เกิดข้อผิดพลาดภายในเซิร์ฟเวอร์
500 EXTERNAL_SERVER_ERROR เซิร์ฟเวอร์ภายนอกขัดข้อง กรุณาลองใหม่ภายหลัง
500 EXTERNAL_INVALID_RESPONSE เซิร์ฟเวอร์ภายนอกตอบกลับมาไม่ถูกต้อง
